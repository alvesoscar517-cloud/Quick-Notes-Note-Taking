# Quick Notes Website

A modern, minimalist website for Quick Notes - AI-Powered Note Taking browser extension.

## Features

- **Modern Design**: Clean, minimalist interface with gold-white gradient theme
- **Responsive**: Fully responsive design that works on all devices
- **Lucide Icons**: Beautiful icons from lucide.dev
- **Smooth Animations**: Subtle fade-in animations and smooth scrolling
- **SEO Optimized**: Proper meta tags and semantic HTML

## Pages

- **index.html**: Main landing page with features, how it works, and download sections
- **privacy.html**: Comprehensive privacy policy
- **policy.html**: Detailed terms of service

## Design System

### Colors
- Gold Light: `#ffd700`
- Gold Medium: `#ffb700`
- Gold Dark: `#ff9500`
- White: `#ffffff`
- Off White: `#fafafa`

### Typography
- System fonts for optimal performance
- Font sizes: 56px (hero), 42px (section headers), 20px (body)

### Components
- Navigation bar with mobile menu
- Hero section with CTA buttons
- Feature cards grid
- Step-by-step guide
- Screenshot showcase
- Download section
- Footer with links

## File Structure

```
website/
├── index.html          # Main landing page
├── privacy.html        # Privacy policy page
├── policy.html         # Terms of service page
├── styles.css          # All styles
├── script.js           # JavaScript functionality
├── icon/
│   └── icon128.png     # App icon
└── image/
    ├── unnamed.png     # Hero image
    ├── unnamed (1).png # Step 1 image
    ├── unnamed (2).png # Step 2 image
    ├── unnamed (3).png # Step 3 image
    └── unnamed (4).png # Screenshot image
```

## Usage

1. Open `index.html` in a web browser
2. All assets are loaded from CDN (Lucide icons)
3. Images are loaded from local folders

## Customization

- Edit colors in `:root` CSS variables in `styles.css`
- Update content in HTML files
- Replace images in `image/` folder
- Modify animations in `script.js`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
